VINCULO — Pop-up de MFA opcional apos o login

Como funciona:
 - Ao entrar (email+senha), se o usuario NAO tem MFA, abre um pop-up
   perguntando se quer ativar.
 - Botoes: "Ativar MFA" (mostra QR + confirma o codigo + codigos de
   recuperacao) e "Lembrar daqui 30 dias" (nao mostra de novo por 30 dias).
 - Quem JA tem MFA nao ve o pop-up e continua sendo desafiado normalmente.

5 arquivos. Sem migracao de banco.

IMPORTANTE: contem tanto a API (auth.ts) quanto o front. Este pacote INCLUI
a mudanca de "MFA opcional" da etapa anterior (auth.ts atualizado).

CMD, na pasta do repo:
  git add .
  git commit -m "Pop-up MFA opcional"
  git push
Publique a API tambem:  cd apps\api  &&  npm run deploy
Teste em aba anonima (Ctrl+Shift+R).
