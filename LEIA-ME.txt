VINCULO — Widget discreto no topo

Mostra no canto superior: Cidade (pelo IP) . Temperatura . Data e Hora de Brasilia.
 - Cidade e temperatura: vem da API (geo do Cloudflare + Open-Meteo, sem chave).
 - Data/hora: fuso America/Sao_Paulo, atualiza sozinho.
 - Aparece no topo do Workspace e do painel Super Admin.
 - Em telas estreitas ele some (discreto).

7 arquivos (2 novos). MUDA A API (nova rota /meta). Sem migracao.

CMD, na pasta do repo:
  git add .
  git commit -m "Widget topo (cidade/hora/clima)"
  git push
Publique a API (obrigatorio, tem rota nova):  cd apps\api  &&  npm run deploy
Teste em aba anonima (Ctrl+Shift+R).

OBS: a cidade e a temperatura sao baseadas no IP de quem acessa
(detectado pelo Cloudflare). Em rede corporativa/VPN pode mostrar a
cidade do servidor de saida, nao a fisica.
